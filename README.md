## Background
[Types](https://github.com/ystr/ystr.github.io) This is the world's most well-known open-source tool for customizing file types..

<img alt="Types" src="https://github.com/user-attachments/assets/7400a74a-b647-407f-9302-b4b1eed820cb" style="width:30%" />

This project aims at providing a mordern appearance for Types. The new design follow the guide of [Fluent Design System](https://www.microsoft.com/design/fluent/)，with flexible，adapting layout，and charming visual design.
